

挂BP版开源地址

1. 安装环境

 [python3.10](python.org)
 [redis](https://github.com/tporadowski/redis/releases)
 [mongodb](https://www.mongodb.com/try/download/community)

创建虚拟环境
 ```shell
  python -m venv . 
```
安装依赖
```shell
  python -r install requirements.txt
```

2. 运行

 启动服务端
 ```sh
  python server/main.py
```

启动客户端

```sh
  python main.py
```
